# About HongFah Online
HongFah Online in Thai is "ห้องฟ้าออนไลน์" is a Java program developed using Object-Oriented Programming techniques that support created and summit online education document, ask question and q&a with HongFah's Staff and also support class attendance check-in we called "HongFah Check-In" feature.

# Tools & Technology Stacks
- **Front-end**: Java Swing
- **Back-end**: Firebase (Cloud Firestore Database)
